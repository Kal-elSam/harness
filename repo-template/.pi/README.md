# Pi Adapter

Pi is an optional CLI adapter.

It is not governance-primary.

## Source of truth

- `AGENTS.md`
- `docs/ai/`
- `docs/skills/`
- `docs/specs/`
- `evals/`

## Use

Use Pi prompt templates and skills to execute the same SDD/TDD/evals/checkpoint workflows from terminal.

## Rule

If Pi instructions conflict with the universal core, the core wins.
