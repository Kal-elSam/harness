# Security

## Reglas no negociables

- No commitear secretos.
- No exponer tokens en logs.
- No copiar valores reales de `.env`.
- Validar input en fronteras de sistema.
- Validar response de APIs externas.
- Aplicar principio de mínimo privilegio.
- Revisar authz además de authn.
- Documentar riesgos de seguridad en specs.
