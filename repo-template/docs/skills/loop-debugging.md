# Loop Debugging

## Cuándo usar

Cuando un loop se atasca, repite fixes, incrementa errores o no converge.

## Procedimiento

1. Detener el loop si alcanzó límites.
2. Revisar errores repetidos.
3. Comparar diff entre iteraciones.
4. Identificar si cambió el scope.
5. Revisar tests/evals fallidas.
6. Escalar a Debugger/modelo fuerte si aplica.
7. Proponer rollback o cambio mínimo.
8. Documentar causa raíz.
