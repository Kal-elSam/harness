# Checkpoint Skill

Use this skill before finalizing work.

## Source of truth

- `AGENTS.md`
- `docs/ai/git-workflow.md`
- `docs/ai/harness.md`

## Workflow

1. Review diff.
2. Check secrets and accidental files.
3. Run lint.
4. Run format.
5. Run typecheck.
6. Run tests.
7. Run evals if applicable.
8. Run build.
9. Propose conventional commit.
10. Wait for human approval.
