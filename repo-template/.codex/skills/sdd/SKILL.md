# SDD Skill

Use this skill for Spec-Driven Development.

## Source of truth

- `AGENTS.md`
- `docs/ai/spec-driven-development.md`
- `docs/ai/harness.md`
- `docs/specs/`

## Workflow

1. Read `AGENTS.md`.
2. Read `docs/ai/spec-driven-development.md`.
3. Create or update the spec in `docs/specs/`.
4. Define acceptance criteria.
5. Create technical plan.
6. Break work into implementation tasks.
7. Do not implement until scope is clear.
