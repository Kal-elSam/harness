# Gentle AI Model Routing

## Economic model profile

Use for:

- intake
- scaffolding
- simple TDD
- docs
- mechanical refactors

## Strong model profile

Use for:

- architecture
- spec review
- eval strategy
- security review
- acceptance review

## Rule

Never allow a low-cost model to approve high-impact architecture, security, auth, payment, data-loss or infra changes without review.
